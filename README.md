# Xnalaye

A clean, responsive portfolio-style website built with HTML, CSS, and assets in the `assets/` directory.

## Overview
Xnalaye showcases projects, visuals, and profile highlights in a simple static site layout.

## Features
- Responsive single-page layout
- Dedicated assets folder for images and icons
- Simple HTML structure for easy customization

## Tech Stack
- HTML5
- CSS3

## Getting Started
Open `index.html` in your browser.

## Folder Structure
- `index.html`
- `assets/`
- `assets/images/`
